# webpack打包自动生成zip压缩包的插件

## Install
```javascript
npm install --save-dev @memory13/webpack-build-compress-plugin
```
## Use
```javascript
const WebpackBuildCompressPlugin = require('@memory13/webpack-build-compress-plugin')

module.exports = {
    plugins: [
        new WebpackBuildCompressPlugin()
    ]
}
```
